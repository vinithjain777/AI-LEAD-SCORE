# AI Lead Score Quiz

A fully interactive lead-scoring quiz for flooring contractors to evaluate their pipeline health.

## What It Does

This quiz assesses contractors across 7 key dimensions:
- Lead Generation sourcing
- Speed to respond to leads
- Follow-up process
- Retargeting
- Reviews & reputation
- Tracking & analytics
- Pipeline consistency

Results include:
- Personalized score (0-100)
- Category breakdown with performance ratings
- Actionable insights and recommendations
- Lead capture form for strategy calls

## Features

- 📊 Real-time progress tracking
- 🎯 Intelligent scoring algorithm
- 🌓 Light/dark theme toggle
- 📱 Fully responsive design
- ⚡ No external dependencies
- 🎨 Modern, accessible UI

## How to Use

Simply open `index.html` in your browser, or deploy to Vercel/Netlify.

## Deployment

This project is configured for Vercel deployment. Just connect your GitHub repository to Vercel and it will deploy automatically on each push.
